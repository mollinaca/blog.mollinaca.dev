A - 深さ優先探索

次の問題を解く。A - 深さ優先探索
実装

H,W = map(int,input().split())
grid = [list(input()) for i in range(H)]

# スタート地点を探す
found = False
for y,line in enumerate(grid):
    for x,p in enumerate(line):
        if p == "s":
            start = (y,x)
            found = True
            break
    if found:
        break

from collections import deque
q = deque()
q.append(start)
direction = [[-1,0],[1,0],[0,-1],[0,1]]
while q:
    p = q.pop() # DFS
    y = p[0]
    x = p[1]
    for d in direction:
        ny = y + d[0]
        nx = x + d[1]

        if 0 <= ny < H and 0 <= nx < W: # gridの範囲内
            
            if grid[ny][nx] == 'g': # ゴールにたどり着いた
                print ('Yes')
                exit ()

            if grid[ny][nx] == '#': # 壁だった
                continue
            else:
                grid[ny][nx] = '#' # 探索済みの場所を壁にする
                q.append((ny,nx)) # 探索キューに積む

print ('No') # たどり着けなかった

ポイント

グラフを取得する grid について

ここまで、隣接行列と隣接リストで取得する方法を書いたが、DFS/BFSにありがちなこの迷路問題の場合は、隣接リストを個人的によく用いる。また graph ではなく grid という変数名にしている。この grid を扱うときは、

  • 左上を原点 (0,0) とし
  • 縦軸下方向に y 軸
  • 横軸右方向に x 軸

とすると、grid[y][x] でそのまま座標へアクセスできるので扱いやすい。
たまに問題によっては左上の原点を(1,1)とする場合があるので、この場合は座標の値を -1 して扱って、解答に（座標を答えるなら） +1 すれば良い。

スタートとゴールを探索する

問題で与えてくれればいいのだけれども、与えてくれない場合もあるので探索する。これはシンプルな全探索。

found = False
for y,line in enumerate(grid):
    for x,p in enumerate(line):
        if p == "s":
            start = (y,x)
            found = True
            break
    if found:
        break

enumerate を使うことで、座標の値をそのまま取得できる。
発見したら Found を True にすることで、処理を終了できる。

directionについて

grid の場合は、上下左右の4方向が基本的な探索方向になる。※頂点を経路で結ぶgraphはここが不規則になるので、このような grid の処理は向かない
すべての座標について4方向を探索するのは、事前に directionで4方向を定義して for 文で回すのが良い。

というわけで、上記コードでACできた。

C - 幅優先探索

C - 幅優先探索

この問題は、スタートとゴールの座標を予め与えてくれるが、原点が (1,1) になっているところが注意。
実装

R,C = map(int,input().split())
sy,sx = map(int,input().split())
gy,gx = map(int,input().split())
sy -= 1
sx -= 1
gy -= 1
gx -= 1
grid = [list(input()) for i in range(R)]
grid2 = [[-1]*C for i in range(R)]
start = (sy,sx,0)
grid2[sy][sx] = start[2]
direction = [[-1,0],[1,0],[0,-1],[0,1]]

from collections import deque
q = deque()
q.append(start)

while q:
    p = q.popleft() # BFS
    y = p[0]
    x = p[1]
    dist = p[2]

    for d in direction:
        ny = y + d[0]
        nx = x + d[1]
        if 0 <= ny < R and 0 <= nx < C: # gridの範囲内
            
            if grid[ny][nx] == '.' and grid2[ny][nx] == -1: # 未探索
                grid2[ny][nx] = dist+1
                q.append((ny,nx,dist+1))
            
            if ny == gy and nx == gx:
                print (grid2[gy][gx])
                exit ()

問題が親切で、上下左右がすべて壁になっているためgridの範囲内かどうかの判定はなくても良い。
汎用的にはあったほうが良いし毎回書いてるので今回も書いている。書いてあっても害はない。

grid2 について

今回は「最短距離」を求める必要があるため、距離を記録している。
具体的には grid と同じ大きさの2次元リストを用意し、調査した距離をそこに記録していくという手法をとっている。たぶんいろんなやり方があると思う。

BFSしているのをDFSに変えてみる

深さ優先探索では、最短距離が測れないのがわかる。

R,C = map(int,input().split())
sy,sx = map(int,input().split())
gy,gx = map(int,input().split())
sy -= 1
sx -= 1
gy -= 1
gx -= 1
grid = [list(input()) for i in range(R)]
grid2 = [[-1]*C for i in range(R)]
start = (sy,sx,0)
grid2[sy][sx] = start[2]
direction = [[-1,0],[1,0],[0,-1],[0,1]]

from collections import deque
q = deque()
q.append(start)

while q:
    p = q.pop() # DFS
    y = p[0]
    x = p[1]
    dist = p[2]

    for d in direction:
        ny = y + d[0]
        nx = x + d[1]
        if 0 <= ny < R and 0 <= nx < C: # gridの範囲内
            
            if grid[ny][nx] == '.' and grid2[ny][nx] == -1: # 未探索

                q.append((ny,nx,dist+1))
            
            if ny == gy and nx == gx:
                print (grid2[gy][gx])
                exit ()
