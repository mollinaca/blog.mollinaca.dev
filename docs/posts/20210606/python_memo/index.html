Python の3項演算子に関するメモ

たまに使うけどこういうときどうすんだっけがわからなくて調べたのでメモしておく。

基本

if n == 1:
    print ('a')
else:
    print ('b')

は、

print ('a') if n == 1 else print ('b')

と書ける。ワンライナーで書いたほうがすっきりする、可読性が高まるときに使う。

elif を使いたい場合

if n == 1:
    print ('a')
elif n == 2:
    print ('b')
else:
   print ('c')

は、

print ('a') if n == 1 else print ('b') if n == 2 else print ('c')

というふうに、 elif を else if と書き直して使う。

場合分けで変数を代入したい場合

if n == 1:
    x = 'a'
elif n == 2:
    x = 'b'
else:
    x = 'c'

これは、

x == 'a' if n == 1 else 'b' if n == 2 else 'c'

と書く。2回目以降の x = は書かない。
