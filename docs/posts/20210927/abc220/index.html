A - Find Multiple

問題: A - Find Multiple

A以上B以下であるCの倍数を一つ出力する。
条件を満たすものがない場合は -1 を出力する。

A,B,C = map(int,input().split())

for x in range(A,B+1):
    if x%C == 0:
        print (x)
        exit ()

print (-1)

いくつか解放を思いつくが、A問題だし単純に for ループを書いてしまうのが早いと思った。

B - Base K

問題: B - Base K

整数A,BがK進数として与えられる。
A×Bを10進数表記で出力する。

K = int(input())
A,B = map(int,input().split())
print (int(str(A),K)*int(str(B),K))

C - Long Sequence

問題: C - Long Sequence

長さNの正の整数からなる文字列Aをいっぱい連結したものを用意し、どのタイミングでその和がXを超えるか？

N = int(input())
A = list(map(int,input().split()))
sA = sum(A)
X = int(input())

# 累積和
W = []
for i,a in enumerate(A):
    if i == 0:
        W.append(a)   
    else:
        W.append(W[i-1]+a)

X2 = X//sA
X3 = X-(X2*sA)

import bisect
print (N*X2+bisect.bisect_right(W,X3)+1)

Aの全部の合計値と、Aの累積和を用意しておき、

  • 合計値の塊が何回目で超えるか？ → その手前に答えがある
  • 累積和の何番目で超えるか？ → 答え と求めた。累積和の何回目？を二分探索で書いたが、ここは愚直にO(N)でやっても問題なさそう。

D - FG operation

問題: D - FG operation

FG2つのオペレーションをたくさん繰り返した全部のパターンを調べてね 以下の回答を作成したが、サンプルはACなものの、RE/TLEとなる。

import copy
from collections import deque
N = int(input())
A = deque(list(map(int,input().split())))

ANS = {}
for i in range(10):
    ANS[i] = 0

def op_f (q:deque) -> deque:
    x = q.popleft()
    y = q.popleft()
    q.appendleft((x+y)%10)
    return q

def op_g (q:deque) -> deque:
    x = q.popleft()
    y = q.popleft()
    q.appendleft((x*y)%10)
    return q

def dfs (q:deque):
    if len(q) == 1:
        ANS[q[0]] += 1
        return
    q_back = copy.deepcopy(q)
    dfs (op_f(q))
    q = copy.deepcopy(q_back)
    dfs (op_g(q))

dfs (A)

for a in ANS.values():
    print (a%998244353)
