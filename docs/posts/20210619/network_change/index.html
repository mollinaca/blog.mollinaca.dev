回線を切り替えた

インターネットに接続する回線とISPを変更した。
変更理由は大したものではなくて、最近は家庭内ポイント利用をdポイントにまとめているため、回線もドコモ光に変更したらお得そう、というだけ。
現状に不満があったわけではないので、悪化するリスクはあったけども、まぁやってみないとわからんというのと、多少悪化してもそこまでじゃないっしょっていう軽いノリで実施。

  • 変更前

    • 回線: NTTフレッツ光
    • ISP: OCN

  • 変更後

    • 回線: ドコモ光
    • ISP: GMOBB

結果

ご覧の通りネットワーク品質としては顕著に悪化。
とはいえ、通常時は遅くても150Mbps程度でてるのでまぁ許容。変更前はだいたい300Mbps～500Mbps程度でてたものが、200Mbps～300Mbpsぐらいになった印象。
ただ、帯域以外にパケロスもちょっと増えてるのでそっちは懸念かなー。ゲームに致命的な影響がなければいいけど。まだそこまでは実感していない。

赤線が切り替えタイミング。
顕著に response time と packet loss が悪化した。

v6プラスについて

よくわかってなかったのでいろいろ調べた。
ネットワークイネイブラー社が公開してる以下の書籍（ラムダノート）がとてもわかりやすかった。

徹底解説v6プラス

IPv6をいい感じに国内で使うためのサービスであるということは理解した上で、利用に制約がでることも了解。特にポート制限がでるのは辛い。
IPSecVPNも使えなくなる（SSL-VPNならいけるが）ので、これはちょっと仕事に支障をきたしそうなので使わないことにした。

念の為、GMOに問い合わせを行い「v4(PPPoE)とv6プラス(IPoE)の併行利用ってできますか？」と確認したが、「大元のNTT回線が2回線引けてればいける」ということだったので、そこまでしてじゃねーなということで採用見送り。
v6プラスが使えれば、品質の悪化を凌駕する高品質ネットワークを手に入れられる可能性もあったと思うけど、今回はいったんこれで妥協することにした。
